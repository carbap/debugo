Just Go

About this project


Just Go is an interactive online playground for the Go programming language, powered by Yaegi, an open-source Go interpreter and debugger.


The backend of this application is a Go program that was compiled to WebAssembly and exposes Yaegi APIs to the browser as JavaScript functions. This allows you to run, debug, set breakpoints, and continue code execution entirely on the client side, without ever communicating with a server!


You can find out more information in the GitHub repository.


Have fun testing out Go!
