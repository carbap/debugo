About this project


Just Go is an interactive playground for the Go programming language, powered by Yaegi, an open-source Go interpreter and debugger.


The backend of this application is a Go program that was compiled to WebAssembly and exposes Yaegi APIs to the browser as JavaScript functions. This allows you to run and debug code entirely on the client side!


Check out the GitHub repository to learn more.


Have fun testing out Go!
